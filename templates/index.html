Available Commands

Club Management
  • /create_club - Create a new club
  • /delete_club - Delete a club
  • /list_clubs - List all clubs
  • /update_club_budget - Update club budget
  • /club_stats - Show club statistics
Player Management
  • /add_player - Add a player
  • /remove_player - Remove a player
  • /update_player_value - Update player value
  • /transfer_player - Transfer player
  • /list_players - List players
Match System
  • /create_match - Schedule a match
  • /list_matches - List upcoming matches
  • /upload_image - Upload match images
Statistics & Utilities
  • /top_players - Top players by value
  • /richest_clubs - Richest clubs
  • /transfer_history - Transfer history
  • /bot_info - Bot information
  • /reset_data - Reset all data
